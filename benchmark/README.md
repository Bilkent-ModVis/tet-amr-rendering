The following scripts reproduce the benchmark results reported in the publication. The scripts expect the renderer
binaries to be available in `../build/`.

| Script                              | Description                                           |
|-------------------------------------|-------------------------------------------------------|
| `benchmark.py`                      | Runs the benchmarks used for Tables 1, 2, 5, and B.6. |
| `mri_tf_opacity_steps_benchmark.py` | Runs the benchmarks used for Table 3.                 |
| `bvh_levels_benchmarks.py`          | Runs the benchmarks used for Table 4.                 |
| `refinement_threshold_benchmark.py` | Runs the benchmarks used for Figures 7 and 8.         |
